Installation

1. clone repo
2. cd to repo Dir
3. create venv
4. activate venv
5. pip install --upgrade pip
6. Run pip install -r requirments.txt

Usage

1. DEPRECATED --- from "<cwd>/tensorflow_object_counting_api" run "python ".../frontend/app.py" ---
1. Run "python main.py"
2. browse to "localhost:5000"
